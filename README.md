## 📸 لمحة سريعة

تطبيق **رحلة التعلم** هو أداة شاملة لتنظيم وتتبع تقدمك في التعلم. يعمل كتطبيق ويب تقدمي (PWA) بدون حاجة لسيرفر، مع إمكانية المزامنة عبر GitHub.

---

## ✨ المميزات

### 📋 إدارة خطط التعلم
- إنشاء خطط متعددة مع مجموعات ومهام
- تتبع التقدم بنسب مئوية وأشرطة
- دعم الدروس الفرعية لكل مهمة
- سحب وإفلات لإعادة الترتيب
- تصنيفات (Tags) وملاحظات لكل مهمة
- تحديد تواريخ مع تنبيهات المهام المتأخرة

### 🛠️ إدارة المشاريع
- مشاريع بمراحل متعددة
- تتبع الحالة: معلق / جاري / مكتمل
- أيقونات مخصصة لكل مشروع

### 🍅 مؤقت بومودورو
- مؤقت Pomodoro مدمج مع إعدادات مخصصة
- تسجيل تلقائي لوقت التعلم
- تنبيهات صوتية ومرئية
- رسم بياني لوقت التعلم الأسبوعي

### 🎯 وضع التركيز (Focus Mode)
- شاشة تركيز كاملة لمهمة واحدة
- بومودورو مدمج مع الدروس الفرعية
- ملاحظات أثناء التركيز
- اختصار: `Alt + F`

### 🧠 البطاقات التعليمية (Flashcards)
- نظام Spaced Repetition (SM-2)
- إنشاء تلقائي من المهام
- جلسات مراجعة تفاعلية
- تقييم: نسيت / صعب / سهل / ممتاز
- اختصار: `Alt + R`

### 🎬 متتبع YouTube
- إضافة قوائم تشغيل YouTube
- تتبع الفيديوهات المشاهدة
- ملاحظات Markdown لكل فيديو
- ربط بخطط التعلم
- اختصار: `Alt + Y`

### 📊 مخطط Gantt
- عرض زمني للمهام والمشاريع
- خط اليوم الحالي
- ألوان حسب الحالة
- اختصار: `Alt + G`

### 🗓️ خريطة النشاط (Heatmap)
- خريطة حرارية سنوية مثل GitHub
- فلاتر: الكل / بومودورو / مهام / يوميات
- إحصائيات مفصلة
- تنقل بين السنوات

### 📸 مشاركة التقدم كصورة
- بطاقة جميلة بإحصائياتك
- 6 خلفيات للاختيار
- تحميل PNG / نسخ / مشاركة
- اختصار: `Alt + S`

### 📝 محرر Markdown
- شريط أدوات كامل
- 3 أوضاع: كتابة / معاينة / تقسيم
- يعمل في الملاحظات واليوميات
- دعم: عناوين، قوائم، كود، جداول، اقتباسات

### 🔗 تتبع الروابط
- ربط مصادر بأي مهمة
- تصنيف تلقائي: فيديو / مقال / كورس / GitHub
- عرض في وضع التركيز

### 📅 التقويم
- عرض شهري للمهام المجدولة
- تصدير إلى Google Calendar
- تصدير ملف .ics
- المهام المتأخرة والقادمة

### 📖 المصادر والمراجع
- تنظيم المصادر في مجموعات
- تتبع حالة كل مصدر
- روابط مباشرة

### 📝 اليوميات
- كتابة يومية بصيغة Markdown
- قوالب جاهزة (تعلم / مراجعة)
- بحث في الإدخالات السابقة

### 🏆 نظام الإنجازات
- 21 إنجاز متنوع
- فتح تلقائي عند تحقيق الشروط
- تنبيهات فورية

### 🌐 دعم اللغات
- عربي + English
- تبديل فوري
- اختصار: `Alt + L`

### 📤 تصدير البيانات
- Obsidian (Markdown)
- Notion (CSV)
- JSON (نسخة احتياطية)

### 🔄 المزامنة
- مزامنة عبر GitHub (مجاني)
- دمج ذكي للبيانات من أجهزة متعددة
- عمل بدون إنترنت (PWA)
- نسخ احتياطية تلقائية يومية

## 🚀 التثبيت والاستخدام

### الطريقة 1: مباشرة (بدون سيرفر)
1. حمّل الملفات أو انسخ الـ Repository
2. افتح `index.html` في المتصفح
3. اختر "تخطي" للحفظ المحلي، أو أعدّ GitHub للمزامنة

### الطريقة 2: GitHub Pages (مع مزامنة)
1. أنشئ Repository جديد على GitHub
2. ارفع كل الملفات
3. فعّل GitHub Pages من Settings
4. افتح الرابط: `https://username.github.io/repo-name`

### الطريقة 3: كتطبيق PWA
1. افتح التطبيق في Chrome/Edge
2. اضغط "تثبيت" أو "Add to Home Screen"
3. يعمل بدون إنترنت!

---

## ⚙️ إعداد المزامنة عبر GitHub

1. اذهب إلى [GitHub Token Settings](https://github.com/settings/tokens/new)
2. اكتب اسم: `learning-journey`
3. اختر صلاحية: `repo` (كامل)
4. اضغط **Generate token** وانسخ الـ Token
5. في التطبيق، أدخل: Username + Repository Name + Token
6. اضغط **بدء الرحلة**

## 🛠️ التقنيات المستخدمة

- **HTML5 / CSS3 / JavaScript** — بدون frameworks
- **PWA** — Service Worker + Manifest
- **GitHub API** — للمزامنة
